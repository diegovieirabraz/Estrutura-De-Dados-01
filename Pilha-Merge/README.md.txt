# 📊 Pilha de Números Pares e Ímpares (Vetor e Lista Encadeada)

## 🧾 Descrição

Este projeto consiste em um programa em C que permite ao usuário digitar **30 números inteiros** de forma crescente. Após a digitação:

- Os números são separados e empilhados em duas estruturas distintas:
  - Uma pilha para números **pares**
  - Uma pilha para números **ímpares**
- As pilhas são implementadas de duas formas:
  - 📦 **Vetor** com capacidade máxima de 30 elementos
  - 🔗 **Lista Encadeada**
- Por fim, os números são desempilhados e exibidos na tela em **ordem decrescente**.

---

## 🎯 Objetivos

- ✅ Praticar a manipulação de pilhas em C
- ✅ Implementar pilhas usando vetor e lista encadeada
- ✅ Aplicar lógica de ordenação crescente e decrescente com pilhas
- ✅ Usar funções como `malloc`, `free`, e ponteiros

---

## 🛠️ Estrutura do Projeto

- `pilha_vetor.c` — Implementação da lógica usando **vetor**
- `pilha_lista.c` — Implementação da lógica usando **lista encadeada**
